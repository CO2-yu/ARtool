# UIフロー

## 通常3Dビュー

URL:

- `/viewer?package=xxx`

表示:

- 製品名
- 製品説明
- `model-viewer`
- スケールスライダー
- 現在倍率
- アニメーション操作
- 「マーカーARで表示」

操作:

- ピンチでズーム
- ドラッグで回転
- スライダーでスケール変更
- アニメーション再生/停止
- ARビューへ遷移

## ARビュー

URL:

- `/ar?package=xxx`

初期表示:

- カメラ許可案内
- マーカー案内
- 最小限の操作UI

マーカー認識後:

- 製品情報パネルを画面下部に表示する。
- 最後に認識したマーカーのpackageIdを `selectedPackageId` とする。

情報パネル:

- 製品名
- 簡易説明
- 「3Dビューで見る」
- 「ARで表示する」

## ボタン動作

### 3Dビューで見る

`/viewer?package=selectedPackageId` へ遷移する。

### ARで表示する

現在選択中のpackageIdのGLBをAR空間へロードして表示する。

## 通常ユーザーとデバッグ

通常ユーザーには詳細な内部情報を表示しない。デバッグ情報は `?debug=1` または `localStorage` flag が有効な場合のみ表示する。
